Point cloud research at NCI

pointWPS

A data service testbed for massive point datasets - using server-side compute to deliver on-demand data subsets and derived products.

Check available services at the the pointwps endpoint; or read the documentation here: how to use the pointwps service

Caution - this is a development service and may be broken at any time. Initial public test release is scheduled for mid-August 2017. Due to storage hardware updates, underlying data may also be somewhat mobile until that time.

Point data visualisation

  • Mh370 phase 1
    A massive bathymetric survey rendered using Potree. See the background story on the search for MH370 at Geoscience Australia's story map. Links for data download can be found here: http://marine.projects.ga.gov.au/mh370-phase-one-data-release.html
  • Canberra in 3D tiles
    This is a 3D point cloud exploration tool using Cesium 3D tiles built by entwine. There are 4.7 billion points in this dataset, which load depending on your viewpoint. Please explore, choose different basemaps, and enjoy! Unfortunately this version is static - we can't turn point classes on or off, or use different basemap data to colour them. It's an initial foray into data exploration tools.

Caution - these are exploratory visualisations and may be broken at any time.
